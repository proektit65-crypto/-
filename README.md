# Attendance Politech

Full working attendance management system built with Flask.

## Features
- Student and admin login
- Attendance marking
- Admin dashboard
- SQLite database (auto-created)
- Templates and static files included

## Tech Stack
- Python
- Flask
- SQLAlchemy
- SQLite
- HTML / CSS / JavaScript

## How to run the project

```bash
git clone https://github.com/USERNAME/proektit65-crypto.git
cd proektit65-crypto
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python main.py
